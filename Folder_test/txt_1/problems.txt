Problem 1: Reverse String
Write a function that takes a string as input and returns the string in reverse order.

Problem 2: Find Missing Number
You are given an array of integers from 1 to N, except for one missing number.
Write a function to find the missing number in the array.

Problem 3: Count Vowels
Write a function that takes a string as input and returns the number of vowels (a, e, i, o, u) in the string.

Problem 4: FizzBuzz
Write a function that prints the numbers from 1 to N. 
But for multiples of three, print "Fizz" instead of the number, and for the multiples of five, print "Buzz".
For numbers that are multiples of both three and five, print "FizzBuzz".

Problem 5: Longest Substring Without Repeating Characters
Given a string, find the length of the longest substring without repeating characters.


Problem 6: Longest Palindromic Substring
Given a string, find the longest substring that is a palindrome. 
A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward.

Problem 7: Two Sum
Given an array of integers and a target sum, find two numbers in the array that add up to the target sum.
Return the indices of the two numbers.
